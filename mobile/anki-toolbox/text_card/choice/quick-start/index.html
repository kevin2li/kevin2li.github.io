快速入门

提示

此功能需要用户了解一点正则表达式的入门知识，如果您还不了解，可在入门必读中学习。

本软件追求的目标是原始文本材料可以直接制卡，无需修改或尽可能少修改文本，实现无痛制卡。

目前支持三种类型的选择题制卡，分别如下：

类型一：答案位于题干中​

参考文本示例：

答案位于题干内.txt
1、Internet中发送邮件协议是（B ）。
A、FTP B、SMTP　
C、HTTP　 D、POP
2、在 OSI模型中，第 N层和其上的 N＋ l层的关系是 (A)
A、N层为N十1层提供服务
B、N十1层将从N层接收的信息增加了一个头
C、N层利用N十1层提供的服务
D、N层对N＋1层没有任何作用
3、早期的计算机网络是由（D ）组成系统。
A、计算机-通信线路-计算机
B、PC机-通信线路-PC机
C．终端-通信线路-终端
D、计算机-通信线路-终端
4．传输速率为（B）的以太网称为传统以太网
A、1M bps B、10M bps
C、100M bps D、1000M bps
5、Ethernet以太网结点最多可达（B）个
A、512 B、1024
C、2048 D、4098
6、EIA RS232C的电气指标规定，信号“1”的额定电压值是（ B）
A、+12伏 B、-12伏
C、+15伏 D、-15伏

类型二：答案位于选项后​

参考文本示例：

答案位于选项后.txt
1、以下合同计价模式下，（　）方式中承包商所承担的风险最小。
A、固定单价合同
B、可变单价合同
C、固定总价合同
D、成本加酬金合同
【正确答案】 D
【答案解析】 本题考查的是招标采购阶段的管理任务和方法。成本加酬金计价模式下，采用这种合同，承包商利润有保证，价格变化或工程量变化的风险基本都由业主承担。但承包商往往缺乏降低成本的激励，还可能通过提高工程成本而增加自身利润，不利于业主的投资控制。因此承包商的风险最小。参见教材P3。
2、根据项目目标要求，要明确项目（　），应用目标分解、工作分解结构（WBS）等方法制定总体采购计划和采购清单。
A、应用横道图、网络图等统筹各项采购任务和时间上的相互衔接
B、采购哪些工程、服务和物资
C、采购勘察设计服务的内容和数量
D、确定采用何种采购和招标方式
【正确答案】 B
【答案解析】 本题考查的是招标采购阶段的管理任务和方法。选项ACD，属于项目招标采购总体策划规定的内容。参见教材P1。
3、开展采购招标工作的主要依据是（　）。
A、确定采用何种采购和招标方式
B、拟定采购方式
C、提前策划设定招标和合同文件中的重点要求
D、拟定采购组织和管理协调工作安排
【正确答案】 C
【答案解析】 本题考查的是招标采购阶段的管理任务和方法。应根据项目需要，提前策划设定招标和合同文件中的重点要求作为开展采购招标工作的主要依据。参见教材P1。
4、下列不可以作为项目参建方的是（　）。
A、监理单位
B、物资供应单位
C、勘察设计单位
D、消防服务机构
【正确答案】 D
【答案解析】 本题考查的是招标采购阶段的管理任务和方法。合同参建方有建设单位、监理单位、勘察设计单位、施工单位、物资供应单位等。参见教材P2。

类型三：问题答案分开​

有时候问题和答案分别位于两个不同文件中，参考文本示例：

  • 问题文件
  • 答案文件
1.男性，38岁。胸痛、发热3天。既往糖尿病病史5年。查体：T37.6℃，右下肺叩浊，呼吸音
减低。胸片提示右侧胸腔积液。胸腔穿刺抽液化验显示胸水WBC650X10°/L，其中淋巴细
胞90%，葡萄糖3.2mmol/L，该患者最可能的诊断是
A.结核性胸膜炎
B.恶性胸腔积液
C.脓胸
D.类肺炎性胸腔积液

2.女性，28岁。乏力、咽痛、头痛、咳嗽、发热2周。查体：T37.8℃，P96次/分，R20次/分，双
肺呼吸音清，未闻及干湿啰音。胸部X线检查显示双下肺多发小片状浸润影。血清支原体
IgM抗体1：64阳性。阿奇霉素治疗1周后症状无好转，复查胸部X线片无变化。此时患者
应换用的抗菌药物是
A.青霉素
B.红霉素
C.头孢呋辛
D.左氧氟沙星

3.下列不符合扩张型心肌病并发心力衰竭的超声心动图表现是
A.LVEF<40%
B.二尖瓣返流
C.左室壁节段运动减弱
D.左心室舒张末内径增大

4.男性，20岁。渐进性心悸、乏力、消瘦6个月，腹胀、下肢水肿2个月。查体：T37.2℃，P106
次/分，BP90/75mmHg，半卧位，颈静脉怒张，双肺（一），心界向两侧扩大，心律整，心音遥
远，心尖部可闻及2/6级收缩期吹风样杂音，脉搏减弱，肝颈静脉回流征阳性，双下肢凹陷性
水肿（十）。该患者最可能的诊断是
A.心包积液
B.风湿性心瓣膜病
C.扩张型心肌病
D.病毒性心肌炎

5.男性，57岁。2天前因突发胸痛，诊断急性前壁心肌梗死，行急诊PCI术，血管再通成功。为
评估此时患者的心脏功能，下列最有价值的血液生化检查是
A.肌钙蛋白
B.肌酸激酶同工酶
C.脑钠肽
D.乳酸脱氢酶
1. A
解析：本题患者，男性，38岁。胸痛、发热3天，有糖尿病病史，低热，查体提示胸腔积液体征，胸水WBC>500X10°/L，应
考虑渗出液，胸水WBC以淋巴细胞为主、葡葡糖<3.3mmol/L,考虑结核性或恶性胸腔积液，由于本题患者
为中年患者，应诊断为结核性胸膜炎所致胸腔积液。恶性胸腔积液常常见于老年人，多无发热症状。脓胸、
类肺炎性胸腔积液常常表现为高热，胸水WBC>10×10°/L，以中性粒细胞为主，葡萄糖降低，与本题
患者不符。

2. D
解析：本题患者，血清支原体1gM抗体≥1：64，应确诊肺炎支原体肺炎，治疗应以大环内酯类抗生素（阿奇霉素、红
霉素等）为首选。本题患者用阿奇霉素治疗1周后症状无好转，说明本题患者对大环内酯类抗生素不敏感，
因此应换用氟喹诺酮类药物治疗，例如左氧氟沙星、莫西沙星等。肺炎支原体没有细胞壁，青霉素、头孢菌素
等抗生素治疗无效。

3. C
解析：左室壁节段性运动减弱多见于缺血性心肌病。LVEF≤40%、二尖瓣返流和左心室舒张末内径增大均符合扩
张型心肌病并发心力衰竭的超声心动图表现。

4. A
解析：本题患者，心界向两侧扩大，应考虑心包积液、扩张型心肌病，根据心音遥远，可诊断为心包积液，并不是扩张
型心肌病。风湿性心瓣膜病和病毒性心肌炎一般不会出现心界向两侧扩大。

5. C
解析：当存在心力衰竭的时候，心室壁张力增加，脑钠肽（BNP)分泌明显增高，其增高程度与心力衰竭的严重程度
呈正相关。因此，脑钠肽(BNP)是评估心脏功能、诊断心力衰竭的常用指标。肌钙蛋白、肌酸激酶同工酶和
乳酸脱氢酶均是评估心肌梗死的常用指标。
