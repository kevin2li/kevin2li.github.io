PDF页码

此功能可以给PDF文件添加页码或删除页码。

添加页码​

本软件支持给PDF文件添加页码，支持多种页码格式，包括：

  • 1,2,3
  • 1/X
  • 第1页
  • 第1/X页
  • 第1页，共X页
  • I,II,III
  • A,B,C
  • a,b,c
  • ...

此外也支持用户自定义页码格式。

自定义页码格式

自定义格式需要勾选上『自定义页码样式』选项。
