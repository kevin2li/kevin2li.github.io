PDF工具箱教程

📄️ 入门必看

本节介绍使用本软件相关的最基础知识。

🗃️ PDF水印

2 个项目

📄️ PDF书签

本功能菜单提供了与PDF书签管理相关的功能：添加书签、提取书签、转换书签、插入目录等。

📄️ PDF转换

本软件提供了多种与PDF相关的格式转换功能。

📄️ PDF旋转

本功能可以将PDF指定范围的页面进行旋转，支持顺时针旋转90、逆时针旋转90和顺时针旋转180。

📄️ PDF合并

本功能包含两种模式：多文件合并和页合并。

📄️ PDF拆分

本功能包含三种模式：均匀分块、自定义范围和目录拆分。

📄️ PDF插入

本功能包含两种模式：插入空白页和插入其他文件。

📄️ PDF替换

此功能可以将PDF文件B(称为替换PDF)的部分页面替换PDF文件A(称为被替换PDF)的部分页面。

📄️ PDF分割

本功能包含三种模式：均匀分割、页分割和自定义分割。

📄️ PDF组合

本功能可以将多个PDF页面重新组合成一个页面。

📄️ PDF背景

本功能包含两种模式：纯色背景和图片背景。

📄️ PDF重排

本功能可以重新调整页面顺序。

📄️ PDF页码

此功能可以给PDF文件添加页码或删除页码。

📄️ PDF页眉页脚

此功能可以给PDF页面添加或删除页眉页脚。

📄️ PDF裁剪

本功能可以对PDF页面进行裁剪，支持2种裁剪模式：根据矩形注释、根据页边距。

📄️ PDF缩放

本功能可以将PDF页面统一进行缩放，支持三种模式：按比例缩放、缩放到指定大小、自定义长宽。

📄️ PDF延展

此功能可以将原有PDF页面进行延展, 支持空白延展和指定文件延展。

📄️ PDF加解密

首先科普一下，PDF文件的密码可分为两种：打开密码和权限密码。

📄️ PDF提取

此功能可以从PDF文件中提取页面、文本、图片、字体等。

📄️ PDF压缩

此功能可以压缩PDF文件，减小文件大小。

📄️ PDF OCR

此功能可以使用OCR技术识别PDF文件中的文字。

📄️ 双层PDF

此功能可以用来制作双层PDF。
