制卡模式

如果你使用的是本软件内置的Kevin Text Cloze挖空模板，支持遮一猜一、遮全猜一、遮半猜一、遮全猜全、自由猜共5种制卡模式，它们各有特点，灵活使用可以满足各类常见需求。

下面具体分别介绍下这几种模式：

1. 遮一猜一​

只有一个空有遮罩，用户只需要猜测该空下面的内容。

这个模式有两个版本：默认模式和『一空一卡』模式。

默认模式​

每个挖空都在一张卡上展示，可以通过底部按钮切换每个挖空。

效果演示：

遮一猜一演示

一空一卡模式​

提示

如果要开启『一空一卡』需要在制卡时勾选上"一空一卡"。

默认模式每个空都位于一张卡上，不便于把握对每个空的掌握程度，如果使用『一空一卡』模式，每个空会独立成一张卡片，复习粒度更细，每张卡片的复习难度也会更低。

效果演示：

遮一猜一(一空一卡)演示

2. 遮全猜一​

全部挖空都有遮罩，但是其中一个空的遮罩颜色与其他空明显不同，用户只需要猜测该空下面的内容。点击该遮罩可以翻开遮罩。

这个模式同样有两个版本：默认模式和『一空一卡』模式。

默认模式​

每个挖空都在一张卡上展示，可以通过底部按钮切换每个挖空。

效果演示：

遮全猜一演示

一空一卡模式​

默认模式每个空都位于一张卡上，不便于把握对每个空的掌握程度，如果使用『一空一卡』模式，每个空会独立成一张卡片，复习粒度更细，每张卡片的复习难度也会更低。

效果演示：

遮全猜一(一空一卡)演示

3. 遮全猜全​

全部挖空都有遮罩并且颜色相同，用户需要猜测所有空下面的内容。点击任何一个遮罩会一次性翻开所有遮罩。

效果演示：

遮全猜全演示

4. 遮半猜一​

全部挖空都有遮罩，但是其中一个空的遮罩颜色与其他空明显不同，用户只需要猜测该空下面的内容，切换到下一张时，前面猜测过的空会保持翻开状态。

效果演示：

遮半猜一演示

5. 自由猜​

全部挖空都有遮罩并且颜色相同, 用户需要猜测所有空下面的内容, 点击哪个遮罩就翻开哪个遮罩，再次点击还可以还原回遮罩状态。

效果演示：

自由猜演示
