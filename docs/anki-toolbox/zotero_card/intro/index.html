介绍

Zotero介绍​

Zotero是一款开源免费的强大的文献管理工具软件，在学术科研领域非常流行。

下载地址：https://www.zotero.org/download/

Zotero主页截图

为什么使用Zotero​

如果你是科研人员​

Zotero无疑是一款文献管理神器，在文献收集、元信息抓取、笔记记录、引用和参考文献生成、论文写作等环节都有助力，可以极大提高科研效率。

如果你不是科研人员​

虽然Zotero主要为文献管理而设计，但是对于更广大的非科研人员来说，也完全可以把Zotero作为自己的第二大脑，利用其出色的资料收集、分类管理、阅读批注、关联引用等能力，作为一个强大的知识管理工具使用，甚至还可以结合Obsidian、Excalidraw、Anki等外部工具，打造出一个非常舒适的学习工作流，实现知识从输入到输出的完美闭环。

尤其是对于一些考研考公等备考人群，学习资料众多，但由于疏于管理，材料散落各处查找不便，或者只进行了简单的文件夹管理，造成笔记和材料分离以及不同材料之间关联引用能力差，白白在一些非重点上浪费了时间，最终导致学习效率低下。

那么相信通过Zotero的加持，可以把你的学习材料、思考笔记等紧密组织联系在一起，并且可以快速关联引用到其他条目，还可以在Obsidian、Anki等外部应用中添加Zotero回链，不同工具软件之间形成了一个有机整体，随着时间推移，最终将构建出专属自己的强大知识库。

我是如何使用Zotero的​

资料收集​

Zotero中的资料管理最小单元是条目(Item)。

Zotero中包含了非常丰富的条目类型,如:

  • 附件(attachments)：最常见的附件就是PDF文件，用户可以直接将PDF文件拖拽到Zoter对应分类中，后续可以使用Zotero内置的PDF阅读器添加各类批注和笔记等。
  • 网页快照(webpage)：配合Zotero官方的浏览器插件Zotero Connector可以非常方便地将网页保存到Zotero资料库中，并且不必担心原网页链接失效问题，Zotero会自动保存一份网页快照，随时都能查看。支持在Zotero中直接阅读网页添加批注做笔记等。
  • 笔记(notes)：Zotero支持添加单独的笔记类型条目，支持标题、公式、超链接、图片、引用等多种富文本格式。
  • ...

Zotero内置的条目类型有几十种，但是对于一般用户来说使用其中常见的几种类型就足够了。

并且Zotero中的条目是支持嵌套的，意味着你可以在一个父条目下，添加若干个子条目，这允许用户可以非常灵活的组织自己的材料。一个常见的情形就是一个父条目下放PDF附件、笔记、其他网页支持材料等。

资料管理​

  • 分类管理

在Zotero可以轻松创建多级嵌套目录，方便用户分门别类的管理自己的文件。在每个分类下，用户都可以添加各种类型的条目资料。

  • 标签管理

支持给条目添加标签,方便后期快速筛选出想要的内容。并且支持给标签分配颜色、分割标签等高级功能。

  • 强大搜索功能

支持搜索元信息、标签和全文检索，还有强大的高级搜索功能。

Zotero高级检索

阅读批注​

Zotero有自己内置的PDF阅读器，并且支持添加高亮、下划线、笔记、文本、矩形、绘图等多种批注工具，可以满足用户常见的记笔记需求。

Zotero批注

并且每种批注都支持添加评论、添加标签、快速切换颜色等等，自定义程度非常高，灵活运用可以极大提高学习效率。

关联引用​

此功能使得所有条目之间不再是一个个孤岛，不管是在Zotero内部，还是Zotero与外部应用(如Obsidian、Anki等)，都可以建立关联。

Zotero内部关联​

Zotero支持给条目添加相关条目，这样当浏览到有关条目时会自动展示与它相关的其他条目，点击即可跳转到对应条目上。

添加相关条目

例如，给A条目添加相关条目B(只用设置A对B的相关即可，B对A的相关是自动添加的)，那么当浏览到A条目时会自动展示相关条目B，浏览B条目时也会展示相关条目A。

外部应用与Zotero关联​

Zotero有自己的zotero://协议，这意味着，可以在外部应用中添加Zotero的回链链接，点击链接即可自动唤起Zotero打开对应的条目，实现外部应用与Zotero之间的互动。例如：

  • 打开PDF附件并跳转至指定页码
    • 举例：zotero://open-pdf/library/items/WNRDR7JD?page=2&annotation=VULFNL6F
    • 含义：打开Zotero中key为WNRDR7JD的条目，并跳转至第3页(链接中的page参数以0表示第1页)的key为VULFNL6F的注释位置
  • 在Zotero中定位到指定条目
    • 举例：zotero://select/library/items/WNRDR7JD
    • 含义：在Zotero主界面中选中key为WNRDR7JD的条目

这一功能基本上支持插入超链接的软件都支持，例如:

  • 众多笔记软件: Obsidian、思源笔记、Logseq等
  • 办公软件: Word、PPT、Excel、Onenote等
  • 思维导图软件: Xmind等
  • 闪卡软件: Anki等
  • 白板软件: Excalidraw等
  • ...

一些典型的使用案例有：

  • 在Obsidian中添加Zotero回链
Obsidian中跳转Zotero
  • 在Anki中添加Zotero回链
Anki中跳转Zotero

当然这里只是举了部分案例，但是展现了Zotero与外部应用连接的丰富可能性，用户可以根据自己的实际需求来灵活使用这一特性，探索属于自己的工作流。

多端同步​

Zotero是一款跨平台的应用，在Win、Mac、Linux、iOS等都有客户端，设置好同步后，可以随时随地查看自己的资料和笔记，非常方便。

进阶使用​

Zotero本身功能已经很强大，但是为了进一步扩展Zotero的能力，优化用户的学习体验和提高学习效率，本软件还提供了PDF笔记和Zotero制卡功能。

PDF笔记 (联动笔记软件)​

Zotero虽然也提供了PDF标注和添加笔记等功能，但毕竟不是专门的笔记软件。如果可以使用其他专业的笔记软件(如Obsidian等)做笔记，同时又能很方便地与Zotero进行联动，那么就能很好结合两者的优势，实现1+1大于2的效果。

为此本软件提供了PDF笔记功能，可以非常方便地实现『Zotero做资料管理+其他专业笔记软件做笔记』的工作流。

详细教程可参见：PDF笔记

Zotero制卡 (联动Anki)​

在通过Zotero收集和整理了一定的学习资料后，一般会对材料中的重点内容添加批注，加入自己的理解。
为了进一步消化吸收，还希望可以方便地将批注内容导入Anki等闪卡软件中，不断进行回顾和背诵，强化学习效果。

本软件提供了丰富的Zotero转Anki功能，支持挖空题、问答题等题型制卡，支持多种制卡模式。

详细教程可参见： Zotero制卡。
