快速入门

文本挖空​

在制作文本填空题时，首先需要对文本进行挖空。可以使用本软件『MD笔记』菜单的文本编辑器对文本进行挖空。

首先将需要挖空的文本粘贴进编辑器中，然后选中需要挖空的文本会自动弹出一个悬浮窗，里面有一排按钮，点击倒数第二个按钮可以进行挖空， 点击倒数第一个按钮则可以取消挖空。

挖空示例
提示

可以按住Ctrl键选择多处需要挖空的文本，然后点击弹窗按钮一次性挖空。

本软件支持多种挖空语法，默认使用的是[[c1::xx]]。如果你需要使用其他的挖空语法，可以点击右上角的"设置"按钮，在弹出的窗口里选择需要的挖空语法。

设置挖空语法

开始制卡​

填空题制卡支持两种模板：

  • Anki内置填空模板：名为填空题, 使用{{c1::xx}}挖空语法，功能较为单一。
  • 本软件内置填空模板：名为Kevin Text Cloze，使用[[c1::xx]]挖空语法，支持根据编号对挖空分组，支持『遮一猜一、遮全猜一、遮全猜全、遮半猜一、自由猜』等多种制卡模式，遮罩支持随点随开，更推荐使用！

使用『Kevin Text Cloze』模板​

这个模板是本软件内置的默认模板。 使用该模板进行填空题制卡前，需要事先对文本进行挖空，本软件支持将各种挖空语法的文本导入，具体如下：

  • {{c1::xx}}: 这也是Anki内置填空题模板的挖空语法，保证了兼容性
  • [[c1::xx]]: 这是该模板最终使用的挖空语法格式，所有其他挖空语法都会转成这种格式
  • {{xx}}: 这是在其他第三方挖空模板中可能会使用的挖空语法
  • **xx**: 这是Markdown语法中的加粗语法，意味着用户可以把Markdown文件中的加粗内容快速挖空制卡

愿陛下托臣以讨贼兴复之效；不效，则治臣之罪，{{c1::以告先帝之灵}}。若无兴德之言，则责攸之、祎、允等之慢，以彰其咎。陛下亦宜自谋，以咨诹善道，察纳雅言，深追先帝遗诏。臣不胜受恩感激。今当远离，{{c1::临表涕零}}，不知所言！
