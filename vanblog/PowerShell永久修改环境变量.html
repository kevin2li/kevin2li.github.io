                                                                                                                                                                            # з®Җд»Ӣ

                                                                                                                                                                            зҺҜеўғеҸҳйҮҸеӯҳеӮЁж“ҚдҪңзі»з»ҹе’Ңе…¶д»–зЁӢеәҸдҪҝз”Ёзҡ„ж•°жҚ®гҖӮ дҫӢеҰӮпјҢ WINDIR зҺҜеўғеҸҳйҮҸеҢ…еҗ« Windows е®үиЈ…зӣ®еҪ•зҡ„дҪҚзҪ®гҖӮ зЁӢеәҸеҸҜд»ҘжҹҘиҜўжӯӨеҸҳйҮҸзҡ„еҖјпјҢд»ҘзЎ®е®ҡ Windows ж“ҚдҪңзі»з»ҹж–Үд»¶жүҖеңЁзҡ„дҪҚзҪ®гҖӮ

                                                                                                                                                                            PowerShell еҸҜд»Ҙи®ҝй—®е’Ңз®ЎзҗҶд»»дҪ•еҸ—ж”ҜжҢҒзҡ„ж“ҚдҪңзі»з»ҹе№іеҸ°дёӯзҡ„зҺҜеўғеҸҳйҮҸгҖӮ еҖҹеҠ© PowerShell зҺҜеўғжҸҗдҫӣзЁӢеәҸпјҢеҸҜд»ҘеңЁеҪ“еүҚжҺ§еҲ¶еҸ°дёӯиҺ·еҸ–гҖҒж·»еҠ гҖҒжӣҙж”№гҖҒжё…йҷӨе’ҢеҲ йҷӨзҺҜеўғеҸҳйҮҸгҖӮ

                                                                                                                                                                            # ж–№жі•

                                                                                                                                                                            ж¶үеҸҠеҲ°дёүдёӘеҸӮж•°пјҡ

                                                                                                                                                                            • env_name: зҺҜеўғеҸҳйҮҸеҗҚз§°
                                                                                                                                                                            • env_value: зҺҜеўғеҸҳйҮҸзҡ„еҖјпјҢе§Ӣз»ҲеӯҳеӮЁдёәеӯ—з¬ҰдёІпјҢдёҚиғҪдёәз©ә
                                                                                                                                                                            • env_scope: зҺҜеўғеҸҳйҮҸзҡ„иҢғеӣҙ

                                                                                                                                                                            еңЁ Windows дёҠпјҢеҸҜд»ҘеңЁдёүдёӘиҢғеӣҙеҶ…е®ҡд№үзҺҜеўғеҸҳйҮҸпјҡ

                                                                                                                                                                            • и®Ўз®—жңә (жҲ–зі»з»ҹ) иҢғеӣҙпјҡMachine
                                                                                                                                                                            • з”ЁжҲ·иҢғеӣҙ: User
                                                                                                                                                                            • иҝӣзЁӢиҢғеӣҙ: еҢ…еҗ«еҪ“еүҚиҝӣзЁӢжҲ– PowerShell дјҡиҜқдёӯеҸҜз”Ёзҡ„зҺҜеўғеҸҳйҮҸгҖӮ жӯӨеҸҳйҮҸеҲ—иЎЁз»§жүҝиҮӘзҲ¶иҝӣзЁӢпјҢжҳҜд»Һи®Ўз®—жңәе’Ңз”ЁжҲ·иҢғеӣҙеҶ…зҡ„еҸҳйҮҸжһ„йҖ зҡ„

                                                                                                                                                                            :::warning{title="жіЁж„ҸпјҡSystem.Environmentж–№жі•жүҚиғҪж°ёд№…жӣҙж”№зҺҜеўғеҸҳйҮҸ"} еңЁ PowerShell дёӯжӣҙж”№зҺҜеўғеҸҳйҮҸж—¶пјҢжӣҙж”№д»…еҪұе“ҚеҪ“еүҚдјҡиҜқгҖӮ жӯӨиЎҢдёәзұ»дјјдәҺ Windows Command Shell дёӯе‘Ҫд»Өзҡ„иЎҢдёә Set е’ҢеҹәдәҺ UNIX зҡ„зҺҜеўғдёӯзҡ„ Setenv е‘Ҫд»Өзҡ„иЎҢдёәгҖӮ иӢҘиҰҒжӣҙж”№Machine жҲ– User дҪңз”Ёеҹҹдёӯзҡ„еҖјпјҢеҝ…йЎ»дҪҝз”Ё System.Environment зұ»зҡ„ж–№жі•гҖӮ
                                                                                                                                                                            иӢҘиҰҒеҜ№и®Ўз®—жңәиҢғеӣҙ(Machine)зҡ„еҸҳйҮҸиҝӣиЎҢжӣҙж”№пјҢиҝҳеҝ…йЎ»е…·жңүз®ЎзҗҶе‘ҳжқғйҷҗгҖӮ :::

                                                                                                                                                                            # и®ҫзҪ®е’Ңдҝ®ж”№зҺҜеўғеҸҳйҮҸ

                                                                                                                                                                            иҜӯжі•пјҡ

                                                                                                                                                                            [Environment]::SetEnvironmentVariable(env_name, env_value, env_scope)

                                                                                                                                                                            зӨәдҫӢпјҡ

                                                                                                                                                                            # и®ҫзҪ®JAVA_HOMEзҺҜеўғеҸҳйҮҸ
[Environment]::SetEnvironmentVariable('JAVA_HOME', 'C:\Program Files\Java\jdk-11.0.13' , 'Machine')

# и®ҫзҪ®PATHзҺҜеўғеҸҳйҮҸ
[Environment]::SetEnvironmentVariable("PATH", $env:PATH + ";C:\Program Files\Go\bin\", "Machine")

                                                                                                                                                                            # жҹҘиҜўзҺҜеўғеҸҳйҮҸ

                                                                                                                                                                            иҜӯжі•пјҡ

                                                                                                                                                                            # ж–№жі•дёҖпјҡ
$Env:<env_name>

# ж–№жі•дәҢпјҡ
[Environment]::GetEnvironmentVariable(env_name, env_scope)

                                                                                                                                                                            # еҲ йҷӨзҺҜеўғеҸҳйҮҸ

                                                                                                                                                                            иҜӯжі•пјҡ

                                                                                                                                                                            [Environment]::SetEnvironmentVariable(env_name, '', env_scope)

                                                                                                                                                                            зӨәдҫӢпјҡ

                                                                                                                                                                            [Environment]::SetEnvironmentVariable('Foo', '', 'Machine')
